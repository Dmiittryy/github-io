Хитбоксы

Есть несколько видов хитбоксов, и каждый из них отвечает за что-то своё. Для начала давайте разберём хитбоксы обЪектов и позже перейдём к хитбоксам игрока. Что такое вообще хитбокс? Хитбокс (от англ. hit box) в Geometry Dash — определённая площадь объекта, при соприкосновении с которой происходит некое действие.

Хитбоксы ненаклонных обЪектов

Убивающие хитбоксы

Хитбокс шипа ктороый имеет треугольную текстуру на самом деле не треугольный. Представляет он из себя небольшой прямоугольник при соприкосновении с которым мы проигрываем. Часто эта фишка (с неполным хитбоксом) используется в уровнях высокой сложности. Далее такой хитбокс (мнговенно убивающий игрока при любом взаимодействии с ним будет убивать его) я буду изображать красным цветом. Таких обЪектов есть немало , ниже будут некоторые из них.
